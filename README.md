# Google Docs PDF Generator

## Data Model

### Sheet: Clients

| Column       | Description                    |
| ------------ | ------------------------------ |
| ID           | Unique client identifier       |
| Client Name  | Client full name               |
| Service      | Service provided               |
| Amount       | Invoice or service amount      |
| Date         | Service or invoice date        |
| PDF URL      | Link to generated PDF document |

**Example row:**

| ID | Client Name | Service              | Amount | Date       | PDF URL |
| -- | ----------- | -------------------- | ------ | ---------- | ------- |
| 1  | John Smith  | Website Development  | 1500   | 2026-06-17 |         |
